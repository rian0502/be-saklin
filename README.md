# Multi-Outlet Laundry Management System (Back-End API)

A robust, enterprise-grade Back-End API built with **Laravel 12**, **Laravel Sanctum**, and **Spatie Laravel Permission**. This system is specifically architected to power multi-tenant and multi-outlet laundry operations, featuring secure API authentication, dynamic outlet-scoped role management (Teams), real-time machine tracking, live inventory controls, and full order-to-delivery pipelines.

---

## 🚀 Getting Started

Follow these comprehensive steps to clone the repository, install all dependencies (including Authentication and Role Management), and set up your local development environment.

### Prerequisites
Before you begin, ensure you have the following installed on your machine:
* Git
* PHP >= 8.2 (Required for Laravel 12)
* Composer
* MySQL 8.x / PostgreSQL 15+

---

### 📦 Installation & Environment Setup

**1. Clone the Repository** Clone this project to your local directory using Git:
```bash
git clone (https://github.com/rian0502/be-saklin.git)
```
**2. Clone the Repository** Clone this project to your local directory using Git:
```bash
cd be-saklin
```
**3. Install PHP Dependencies Install Laravel core dependencies and the required packages via Composer:
```bash
composer install
```
**4. Environment Configuration
```bash
cp .env.example .env
```
